1. Propiedad Conmutativa +

Definición

El orden de los sumandos no altera el resultado de la suma.

Fórmula

$$a + b = b + a$$

Ejemplo

Si sumamos 3 + 5 o 5 + 3, el resultado es el mismo:

$$3 + 5 = 8$$ $$5 + 3 = 8$$

Aplicaciones

  • Reorganización de términos en expresiones algebraicas
  • Simplificación de cálculos mentales
  • Demostración de teoremas matemáticos
2. Propiedad Asociativa +

Definición

La forma en que se agrupan los sumandos no altera el resultado de la suma.

Fórmula

$$(a + b) + c = a + (b + c)$$

Ejemplo

Al sumar (2 + 3) + 4 o 2 + (3 + 4), el resultado es el mismo:

$$(2 + 3) + 4 = 5 + 4 = 9$$ $$2 + (3 + 4) = 2 + 7 = 9$$

Aplicaciones

  • Simplificación de expresiones algebraicas
  • Algoritmos de cálculo numérico
  • Demostraciones matemáticas
3. Elemento Neutro (Identidad Aditiva) +

Definición

Existe un número que, al sumarlo a cualquier otro, no altera su valor. Este número es el cero (0).

Fórmula

$$a + 0 = a$$ $$0 + a = a$$

Ejemplo

Al sumar cualquier número con cero, el resultado es el mismo número:

$$7 + 0 = 7$$ $$0 + (-3) = -3$$

Aplicaciones

  • Resolución de ecuaciones
  • Demostraciones algebraicas
  • Algoritmos de computación
4. Inverso Aditivo (Opuesto) +

Definición

Para todo número a, existe un número -a tal que su suma es igual a cero (el elemento neutro).

Fórmula

$$a + (-a) = 0$$ $$-a + a = 0$$

Ejemplo

El opuesto de 5 es -5, y su suma es cero:

$$5 + (-5) = 0$$ $$-3 + 3 = 0$$

Aplicaciones

  • Resolución de ecuaciones lineales
  • Simplificación de expresiones algebraicas
  • Definición de la resta como suma del opuesto
