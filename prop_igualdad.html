Propiedad Reflexiva +

Definición

Todo número es igual a sí mismo.

Fórmula

$$a = a$$

Ejemplo

Si tenemos el número 5, entonces:

$$5 = 5$$
Propiedad Simétrica +

Definición

Si un número es igual a otro, entonces el segundo es igual al primero.

Fórmula

$$\text{Si } a = b, \text{ entonces } b = a$$

Ejemplo

Si sabemos que 3 + 2 = 5, entonces:

$$5 = 3 + 2$$
Propiedad Transitiva +

Definición

Si un número es igual a un segundo número y este es igual a un tercero, entonces el primero es igual al tercero.

Fórmula

$$\text{Si } a = b \text{ y } b = c, \text{ entonces } a = c$$

Ejemplo

Si 2 + 3 = 5 y 5 = 10/2, entonces:

$$2 + 3 = \frac{10}{2}$$
Propiedad de Sustitución +

Definición

Si dos cantidades son iguales, una puede ser reemplazada por la otra en cualquier expresión sin cambiar su valor.

Fórmula

$$\text{Si } a = b, \text{ entonces } a \text{ puede reemplazar a } b \text{ en cualquier expresión}$$

Ejemplo

Si x = 5, entonces en la expresión 2x + 3 podemos sustituir x por 5:

$$2(5) + 3 = 10 + 3 = 13$$
Propiedad Aditiva +

Definición

Si sumamos la misma cantidad a ambos lados de una igualdad, la igualdad se mantiene.

Fórmula

$$\text{Si } a = b, \text{ entonces } a + c = b + c$$

Ejemplo

Si x - 3 = 7, podemos sumar 3 a ambos lados:

$$x - 3 + 3 = 7 + 3$$ $$x = 10$$
Propiedad Multiplicativa +

Definición

Si multiplicamos ambos lados de una igualdad por la misma cantidad (distinta de cero), la igualdad se mantiene.

Fórmula

$$\text{Si } a = b \text{ y } c \neq 0, \text{ entonces } a \times c = b \times c$$

Ejemplo

Si x/2 = 4, podemos multiplicar ambos lados por 2:

$$\frac{x}{2} \times 2 = 4 \times 2$$ $$x = 8$$
