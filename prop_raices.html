Propiedades de las Raíces (Radicales)

home Volver al Inicio

Explora las propiedades fundamentales de las raíces. Recuerda: Las raíces son solo exponentes disfrazados.

1. Definición de raíz n-ésima +

Definición

La raíz n-ésima de un número a es un número b tal que b elevado a n es igual a a.

Fórmula

$$\sqrt[n]{a} = b \quad \text{si y solo si} \quad b^n = a$$

Ejemplo

Ejemplos de raíces:

$$\sqrt[3]{8} = 2 \quad \text{porque} \quad 2^3 = 8$$ $$\sqrt{16} = 4 \quad \text{porque} \quad 4^2 = 16$$

Notación

  • Cuando n=2, se llama raíz cuadrada y se omite el índice: $$\sqrt{a} = \sqrt[2]{a}$$
  • Cuando n=3, se llama raíz cúbica
  • Para n≥4, se lee "raíz n-ésima"
2. Raíz de un producto +

Definición

La raíz n-ésima de un producto es igual al producto de las raíces n-ésimas de los factores.

Fórmula

$$\sqrt[n]{a \times b} = \sqrt[n]{a} \times \sqrt[n]{b}$$

Ejemplo

Raíz de un producto:

$$\sqrt{9 \times 4} = \sqrt{9} \times \sqrt{4} = 3 \times 2 = 6$$ $$\sqrt[3]{8 \times 27} = \sqrt[3]{8} \times \sqrt[3]{27} = 2 \times 3 = 6$$

Aplicaciones

  • Simplificación de expresiones con raíces
  • Factorización de radicales
  • Cálculo mental de raíces
3. Raíz de un cociente +

Definición

La raíz n-ésima de un cociente es igual al cociente de las raíces n-ésimas del numerador y el denominador.

Fórmula

$$\sqrt[n]{\frac{a}{b}} = \frac{\sqrt[n]{a}}{\sqrt[n]{b}} \quad (b \neq 0)$$

Ejemplo

Raíz de un cociente:

$$\sqrt{\frac{25}{4}} = \frac{\sqrt{25}}{\sqrt{4}} = \frac{5}{2}$$ $$\sqrt[3]{\frac{64}{8}} = \frac{\sqrt[3]{64}}{\sqrt[3]{8}} = \frac{4}{2} = 2$$

Importancia

  • Simplificación de fracciones con raíces
  • Racionalización de denominadores
  • Resolución de ecuaciones con radicales
4. Raíz de una raíz +

Definición

La raíz m-ésima de una raíz n-ésima es igual a la raíz (m×n)-ésima del radicando.

Fórmula

$$\sqrt[m]{\sqrt[n]{a}} = \sqrt[m \times n]{a}$$

Ejemplo

Raíz de una raíz:

$$\sqrt{\sqrt{16}} = \sqrt[4]{16} = 2$$ $$\sqrt[3]{\sqrt[4]{x^{12}}} = \sqrt[12]{x^{12}} = x$$

Aplicaciones

  • Simplificación de expresiones con raíces anidadas
  • Conversión entre diferentes índices de raíces
  • Resolución de ecuaciones con radicales múltiples
5. Potencia de una raíz +

Definición

Una potencia dentro de una raíz puede expresarse como una raíz con el exponente del radicando.

Fórmula

$$\sqrt[n]{a^m} = a^{\frac{m}{n}} = (\sqrt[n]{a})^m$$

Ejemplo

Potencia de una raíz:

$$\sqrt[3]{8^2} = 8^{\frac{2}{3}} = (\sqrt[3]{8})^2 = 2^2 = 4$$ $$\sqrt{x^6} = x^{\frac{6}{2}} = x^3$$
Nota: Esta propiedad muestra la relación fundamental entre raíces y exponentes fraccionarios.
6. Simplificación de raíces +

Definición

Las raíces pueden simplificarse extrayendo factores del radicando que sean potencias perfectas según el índice.

Fórmula

$$\sqrt[n]{a^{n} \times b} = a \times \sqrt[n]{b}$$

Ejemplo

Simplificación de raíces:

$$\sqrt{50} = \sqrt{25 \times 2} = 5\sqrt{2}$$ $$\sqrt[3]{54} = \sqrt[3]{27 \times 2} = 3\sqrt[3]{2}$$

Pasos para simplificar

  1. Factorizar el radicando en potencias perfectas
  2. Extraer las raíces exactas
  3. Multiplicar los factores fuera de la raíz
  4. Dejar los factores que no son potencias perfectas dentro de la raíz
7. Racionalización de denominadores +

Definición

Proceso para eliminar raíces del denominador multiplicando numerador y denominador por una expresión adecuada.

Casos principales

Caso 1: Denominador con un solo término

$$\frac{a}{\sqrt{b}} = \frac{a \times \sqrt{b}}{\sqrt{b} \times \sqrt{b}} = \frac{a\sqrt{b}}{b}$$

Caso 2: Denominador con binomio

$$\frac{a}{\sqrt{b} + \sqrt{c}} = \frac{a(\sqrt{b} - \sqrt{c})}{(\sqrt{b} + \sqrt{c})(\sqrt{b} - \sqrt{c})} = \frac{a(\sqrt{b} - \sqrt{c})}{b - c}$$

Ejemplo

Racionalizar denominadores:

$$\frac{3}{\sqrt{5}} = \frac{3\sqrt{5}}{5}$$ $$\frac{2}{\sqrt{3} - 1} = \frac{2(\sqrt{3} + 1)}{(\sqrt{3} - 1)(\sqrt{3} + 1)} = \frac{2\sqrt{3} + 2}{3 - 1} = \sqrt{3} + 1$$
Importante: La racionalización es útil para simplificar expresiones y facilitar cálculos posteriores.
