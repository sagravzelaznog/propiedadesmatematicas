1. Linealidad de la integral +

Definición

La integral de una suma es la suma de las integrales, y la integral de una constante por una función es la constante por la integral de la función.

Fórmula

$$\int [a f(x) + b g(x)] \, dx = a \int f(x) \, dx + b \int g(x) \, dx$$

Ejemplo

Ejemplo de linealidad:

$$\int (3x^2 + 2\sin x) \, dx = 3\int x^2 \, dx + 2\int \sin x \, dx = x^3 - 2\cos x + C$$

Aplicaciones

  • Descomposición de integrales complejas en partes más simples
  • Cálculo de áreas compuestas
  • Resolución de ecuaciones diferenciales lineales
2. Teorema Fundamental del Cálculo +

Definición

Establece la relación entre derivación e integración, permitiendo calcular integrales definidas a partir de primitivas.

Fórmula

$$\int_{a}^{b} f(x) \, dx = F(b) - F(a)$$ $$\text{donde } F'(x) = f(x)$$

Ejemplo

Cálculo de una integral definida:

$$\int_{0}^{\pi} \sin x \, dx = [-\cos x]_{0}^{\pi} = (-\cos \pi) - (-\cos 0) = 1 - (-1) = 2$$

Importancia

  • Conecta el cálculo diferencial e integral
  • Permite calcular áreas bajo curvas
  • Base para el teorema de cambio neto
3. Integración por partes +

Definición

Técnica para integrar el producto de dos funciones, basada en la regla del producto para derivadas.

Fórmula

$$\int u \, dv = uv - \int v \, du$$

Ejemplo

Integración por partes de x·eˣ:

$$\text{Sea } u = x, dv = e^x dx \Rightarrow du = dx, v = e^x$$ $$\int x e^x \, dx = x e^x - \int e^x \, dx = x e^x - e^x + C = e^x(x - 1) + C$$

Regla LIATE

Orden de preferencia para elegir u (de mayor a menor preferencia):

  • Logarítmicas
  • Inversas trigonométricas
  • Algebraicas (polinomios)
  • Trigonométricas
  • Exponenciales
4. Sustitución trigonométrica +

Definición

Técnica para integrar funciones que contienen expresiones de la forma √(a² - x²), √(a² + x²) o √(x² - a²).

Sustituciones comunes

$$\text{Para } \sqrt{a^2 - x^2} \quad \text{usar} \quad x = a \sin \theta$$ $$\text{Para } \sqrt{a^2 + x^2} \quad \text{usar} \quad x = a \tan \theta$$ $$\text{Para } \sqrt{x^2 - a^2} \quad \text{usar} \quad x = a \sec \theta$$

Ejemplo

Integral con raíz cuadrada:

$$\int \frac{dx}{\sqrt{9 - x^2}} = \arcsin\left(\frac{x}{3}\right) + C$$
Consejo: Dibujar un triángulo rectángulo puede ayudar a visualizar la sustitución y simplificar la expresión final.
5. Integrales impropias +

Definición

Integrales con intervalos infinitos o con funciones no acotadas en el intervalo de integración.

Tipos

$$\int_{a}^{\infty} f(x) \, dx = \lim_{b \to \infty} \int_{a}^{b} f(x) \, dx$$ $$\int_{-\infty}^{b} f(x) \, dx = \lim_{a \to -\infty} \int_{a}^{b} f(x) \, dx$$ $$\int_{a}^{b} f(x) \, dx \quad \text{donde f tiene una asíntota vertical en } c \in [a,b]$$

Ejemplo

Integral impropia convergente:

$$\int_{1}^{\infty} \frac{1}{x^2} \, dx = \lim_{b \to \infty} \left[-\frac{1}{x}\right]_{1}^{b} = \lim_{b \to \infty} \left(1 - \frac{1}{b}\right) = 1$$
¡Cuidado! No todas las integrales impropias convergen. Es necesario verificar la convergencia antes de calcular su valor.
6. Aplicaciones de la integral +

Áreas entre curvas

$$A = \int_{a}^{b} |f(x) - g(x)| \, dx$$

Volúmenes de revolución

Método de discos:

$$V = \pi \int_{a}^{b} [f(x)]^2 \, dx$$

Método de capas cilíndricas:

$$V = 2\pi \int_{a}^{b} x \cdot f(x) \, dx$$

Longitud de arco

$$L = \int_{a}^{b} \sqrt{1 + [f'(x)]^2} \, dx$$
Nota: Estas son solo algunas de las muchas aplicaciones del cálculo integral en física, ingeniería, economía y otras ciencias.
