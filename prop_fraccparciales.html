Descomposición en Fracciones Parciales

Aprende a descomponer funciones racionales en fracciones más simples para facilitar su integración. Haz clic en cada caso para ver ejemplos y procedimientos detallados.

1. Introducción a las Fracciones Parciales +

¿Qué son las fracciones parciales?

La descomposición en fracciones parciales es una técnica que permite expresar una función racional como la suma de fracciones más simples, facilitando su integración o manipulación algebraica.

Forma general

$$\frac{P(x)}{Q(x)} = \text{Suma de fracciones más simples}$$

Condiciones iniciales

  • El grado del numerador P(x) debe ser menor que el del denominador Q(x). Si no es así, primero se debe realizar la división polinómica.
  • Q(x) debe poder factorizarse en factores lineales y/o cuadráticos irreducibles.
Importante: Esta técnica es especialmente útil para integrar funciones racionales.
2. Caso 1: Factores Lineales Distintos +

Forma general

$$\frac{P(x)}{(x-a_1)(x-a_2)\cdots(x-a_n)} = \frac{A_1}{x-a_1} + \frac{A_2}{x-a_2} + \cdots + \frac{A_n}{x-a_n}$$

Ejemplo

Descomponer: $$\frac{3x+5}{(x-1)(x+2)}$$

Paso 1: Establecer la forma de la descomposición
$$\frac{3x+5}{(x-1)(x+2)} = \frac{A}{x-1} + \frac{B}{x+2}$$
Paso 2: Multiplicar ambos lados por el denominador común
$$3x + 5 = A(x+2) + B(x-1)$$
Paso 3: Resolver para A y B (usando valores estratégicos de x)
$$\text{Si } x = -2: 3(-2) + 5 = B(-2-1) \Rightarrow -1 = -3B \Rightarrow B = \frac{1}{3}$$ $$\text{Si } x = 1: 3(1) + 5 = A(1+2) \Rightarrow 8 = 3A \Rightarrow A = \frac{8}{3}$$
Paso 4: Escribir la descomposición final
$$\frac{3x+5}{(x-1)(x+2)} = \frac{8/3}{x-1} + \frac{1/3}{x+2}$$
3. Caso 2: Factores Lineales Repetidos +

Forma general

$$\frac{P(x)}{(x-a)^n} = \frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + \cdots + \frac{A_n}{(x-a)^n}$$

Ejemplo

Descomponer: $$\frac{x^2 + 2x - 1}{(x+1)^3}$$

$$\frac{x^2 + 2x - 1}{(x+1)^3} = \frac{A}{x+1} + \frac{B}{(x+1)^2} + \frac{C}{(x+1)^3}$$
Solución: A = 1, B = 0, C = -2
$$= \frac{1}{x+1} - \frac{2}{(x+1)^3}$$
4. Caso 3: Factores Cuadráticos Irreducibles +

Forma general

$$\frac{P(x)}{(ax^2 + bx + c)Q(x)} = \frac{Ax + B}{ax^2 + bx + c} + \text{otros términos}$$

Ejemplo

Descomponer: $$\frac{2x^2 - x + 4}{(x^2 + 1)(x - 2)}$$

$$= \frac{Ax + B}{x^2 + 1} + \frac{C}{x - 2}$$
Solución: A = -1, B = 0, C = 2
$$= \frac{-x}{x^2 + 1} + \frac{2}{x - 2}$$
5. Aplicaciones en Integración +

Integración de funciones racionales

La descomposición en fracciones parciales simplifica enormemente la integración de funciones racionales.

Ejemplo

Calcular: $$\int \frac{3x+5}{(x-1)(x+2)} dx$$

Paso 1: Descomponer en fracciones parciales (visto en el Caso 1)
$$= \int \left(\frac{8/3}{x-1} + \frac{1/3}{x+2}\right) dx$$
Paso 2: Integrar cada término por separado
$$= \frac{8}{3} \ln|x-1| + \frac{1}{3} \ln|x+2| + C$$

Consejos para la integración

  • Siempre verifica si el grado del numerador es menor que el del denominador.
  • Para factores cuadráticos en el denominador, completa el cuadrado si es necesario.
  • Recuerda que $\int \frac{1}{x^2 + a^2} dx = \frac{1}{a} \arctan\left(\frac{x}{a}\right) + C$
