1. Multiplicación de potencias de igual base +

Definición

Al multiplicar potencias de la misma base, se suman los exponentes.

Fórmula

$$a^m \times a^n = a^{m+n}$$

Ejemplo

Multiplicar potencias con la misma base:

$$2^3 \times 2^4 = 2^{3+4} = 2^7 = 128$$ $$5^2 \times 5^3 = 5^{2+3} = 5^5 = 3125$$

Aplicaciones

  • Simplificación de expresiones algebraicas
  • Cálculo de intereses compuestos
  • Notación científica
2. División de potencias de igual base +

Definición

Al dividir potencias de la misma base, se restan los exponentes.

Fórmula

$$\frac{a^m}{a^n} = a^{m-n} \quad (a \neq 0)$$

Ejemplo

Dividir potencias con la misma base:

$$\frac{3^5}{3^2} = 3^{5-2} = 3^3 = 27$$ $$\frac{10^7}{10^4} = 10^{7-4} = 10^3 = 1000$$
Nota: Si el exponente del numerador es menor que el del denominador, el resultado tendrá exponente negativo, que equivale a 1 sobre la potencia con exponente positivo.
3. Potencia de una potencia +

Definición

Al elevar una potencia a otra potencia, se multiplican los exponentes.

Fórmula

$$(a^m)^n = a^{m \times n}$$

Ejemplo

Elevar una potencia a otra potencia:

$$(2^3)^4 = 2^{3 \times 4} = 2^{12} = 4096$$ $$(5^2)^3 = 5^{2 \times 3} = 5^6 = 15625$$

Aplicaciones

  • Simplificación de expresiones con exponentes múltiples
  • Cálculo de crecimiento exponencial compuesto
  • Álgebra de funciones exponenciales
4. Potencia de un producto +

Definición

La potencia de un producto es igual al producto de las potencias de los factores.

Fórmula

$$(a \times b)^n = a^n \times b^n$$

Ejemplo

Elevar un producto a una potencia:

$$(2 \times 3)^4 = 2^4 \times 3^4 = 16 \times 81 = 1296$$ $$(5x)^2 = 5^2 \times x^2 = 25x^2$$

Importancia

  • Fundamental para la factorización de expresiones algebraicas
  • Útil en cálculos con notación científica
  • Base para entender raíces de productos
5. Potencia de un cociente +

Definición

La potencia de un cociente es igual al cociente de la potencia del numerador entre la potencia del denominador.

Fórmula

$$\left(\frac{a}{b}\right)^n = \frac{a^n}{b^n} \quad (b \neq 0)$$

Ejemplo

Elevar un cociente a una potencia:

$$\left(\frac{3}{4}\right)^2 = \frac{3^2}{4^2} = \frac{9}{16}$$ $$\left(\frac{2x}{5}\right)^3 = \frac{(2x)^3}{5^3} = \frac{8x^3}{125}$$

Aplicaciones

  • Simplificación de fracciones complejas
  • Resolución de ecuaciones racionales
  • Cálculo de probabilidades
6. Exponente cero +

Definición

Cualquier número distinto de cero elevado a la potencia cero es igual a uno.

Fórmula

$$a^0 = 1 \quad (a \neq 0)$$

Ejemplo

Ejemplos de exponente cero:

$$5^0 = 1$$ $$(-3)^0 = 1$$ $$(\pi)^0 = 1$$
¡Cuidado! La expresión $0^0$ es una forma indeterminada en matemáticas.

Demostración

Usando la propiedad de división de potencias:

$$a^0 = a^{n-n} = \frac{a^n}{a^n} = 1$$
7. Exponente negativo +

Definición

Un exponente negativo indica el recíproco de la base elevada al valor absoluto del exponente.

Fórmula

$$a^{-n} = \frac{1}{a^n} \quad (a \neq 0)$$

Ejemplo

Convertir exponentes negativos a positivos:

$$2^{-3} = \frac{1}{2^3} = \frac{1}{8}$$ $$\left(\frac{3}{4}\right)^{-2} = \left(\frac{4}{3}\right)^2 = \frac{16}{9}$$

Importancia

  • Permite trabajar con cantidades muy pequeñas
  • Fundamental en notación científica
  • Base para funciones exponenciales decrecientes
8. Exponente fraccionario (Raíces) +

Definición

Un exponente fraccionario representa una raíz. El denominador de la fracción indica el índice de la raíz.

Fórmula

$$a^{\frac{m}{n}} = \sqrt[n]{a^m} = (\sqrt[n]{a})^m$$

Ejemplo

Convertir exponentes fraccionarios a radicales:

$$8^{\frac{2}{3}} = (\sqrt[3]{8})^2 = 2^2 = 4$$ $$x^{\frac{1}{2}} = \sqrt{x}$$
Nota: Cuando el exponente es 1/2, se puede escribir simplemente como raíz cuadrada.

Aplicaciones

  • Conversión entre notación de exponentes y radicales
  • Simplificación de expresiones con raíces
  • Resolución de ecuaciones exponenciales y radicales
